# Dataset

The dataset used in this project is not included in the repository due to its size.

You can download the dataset from Google Drive:

**Google Drive**
[Download Dataset](https://drive.google.com/drive/folders/1jvsWNirBeMli9NZxZm0x4_JIgobecuTg?usp=sharing)

After downloading, place the files in this directory using the following structure:

```
data/
├── creditcard_fraud_train.csv
└── creditcard_fraud_test.csv
```

These files are required to run the training and inference pipelines.
